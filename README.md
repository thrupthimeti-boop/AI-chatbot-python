# AI Chatbot in Python

## Description
A simple AI chatbot built using Python. It can answer basic questions, tell jokes, remember the user's name, display the date and time, provide motivational quotes, and perform simple calculations.

## Features
- Greeting conversation
- Date and Time
- Calculator
- Random Jokes
- Remember User Name
- Motivational Quotes
- Help Menu
- Weather Placeholder
- Random Facts

## Technologies Used
- Python 3
- CSV Module
- datetime Module
- random Module

## Files
- main.py
- user.csv
- README.md

## How to Run
1. Install Python 3.
2. Run `main.py`.
3. Start chatting with Chintu.
